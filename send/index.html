Student Learning Message

Prepare a personalised learning message, paste up to three images and share the generated content through WhatsApp.

Create a Message

Select Daily, Weekly or Others. Some parts of the message are selected randomly so students do not receive exactly the same wording every time.

Enter as many links as required. Put each link on a separate line.

Paste or Select Images

Copy an image or screenshot, tap inside a paste box and paste it. You may also select an image file. Each image is converted into a Data URL.

Message Preview

Greetings, headings, encouragement and closing lines are selected randomly. Preview, Copy and Send use the same generated version until the form is edited or a new version is requested.
Image Data URLs can be extremely long. Large images may make the WhatsApp message too long. Small compressed screenshots are more likely to work.
